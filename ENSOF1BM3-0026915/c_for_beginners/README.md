
# Apresentação da Disciplina: Algoritmos e Programação
**Prof. Hudson Neves**

## 1. Boas-vindas à Disciplina
Este repositório reúne o conteúdo, exemplos, atividades e guias de apoio da disciplina **Algoritmos e Programação**. O objetivo é formar uma base sólida de raciocínio lógico e introduzir os fundamentos essenciais da linguagem C.

Ao longo do semestre, exploraremos:
- Lógica de programação
- Estruturas básicas da linguagem C
- Práticas de desenvolvimento limpo e organizado
- Exercícios orientados à resolução de problemas

---

## 2. Sobre o Ambiente de Desenvolvimento: GitHub Codespaces
Para garantir maior produtividade, padronização e facilitar o aprendizado de todos, utilizaremos o **GitHub Codespaces** como ambiente principal de desenvolvimento.

### Por que usar o Codespaces?
- Não requer instalação local de ferramentas
- Ambiente configurado automaticamente
- Funciona em qualquer computador (Windows, Linux, Mac)
- Permite programar diretamente do navegador ou do VS Code
- Facilita o compartilhamento de código e o acompanhamento da evolução do aluno

### O que você precisa?
- Uma conta gratuita no GitHub
- Acesso à internet

---

## 3. Como utilizar o Codespaces

### Passo 1 — Acesse o repositório da disciplina
Você receberá o link no Moodle ou no Teams.

### Passo 2 — Clique em “Code”
Depois selecione a opção **Codespaces**.

### Passo 3 — Crie um Codespace
Clique em:
```
Create codespace on main
```
O ambiente será configurado automaticamente com todas as ferramentas necessárias.

### Passo 4 — Comece a programar
O Codespaces oferece:
- Terminal integrado
- Editor de código VS Code
- Compilador e ferramentas essenciais
